# Jas Gill — GitHub Pages site

## What this is
A fast, premium, no-framework site (HTML/CSS/JS) built for GitHub Pages.

## Quick setup
1. Create a repo named: `<your-username>.github.io`
2. Upload these files to the repo root:
   - index.html
   - projects.html
   - styles.css
   - script.js
3. In GitHub: Settings → Pages → Deploy from a branch → `main` / `(root)`

## Customize in 2 minutes
- Replace placeholder email `jas@example.com`
- Replace LinkedIn URL
- Replace the “Add link” buttons in `projects.html` with real URLs

## Add screenshots
Put images in an `assets/` folder and reference them like:
`<img src="assets/myshot.png" alt="..." />`
